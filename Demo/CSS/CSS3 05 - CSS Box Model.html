Навігація

Що таке специфічність?

Якщо є два або більше правил CSS, які вказують на той самий елемент, селектор із найвищим значенням специфічності «переможе», а його оголошення стилю буде застосовано до цього елемента HTML.

CSS Box Model

Що таке Box Model?

Кожен елемент, який можна відобразити на веб-сторінці, складається з одного або кількох прямокутних блоків. Модель поля CSS зазвичай описує, як ці прямокутні поля розташовані на веб-сторінці. Ці блоки можуть мати різні властивості та можуть взаємодіяти один з одним по-різному, але кожен блок має область вмісту та необов’язкові оточуючі області відступу , кордону та полів .

Приклад:

Пояснення різних частин:

  • Content (Вміст) – вміст поля, де з’являються текст і зображення;
  • Padding (Внутрішній відступ) – очищає область навколо вмісту. Прозоре поле;
  • Border (Рамка) – межа, що охоплює відступи та вміст;
  • Margin (Зовнішній відступ) – очищає область за межами. Прозоре поле;

Демонстрація моделі за допомогою CSS:

          
            div {
              width: 50px;
              height: 50px;
              border: 10px dotted magenta;
              padding: 50px;
              margin: 50px 60px 40px 30px;
            }
