Що таке Selector?

Селектор CSS — це шаблон для відповідності елементів веб-сторінки. Правила стилю, пов’язані з цим селектором, будуть застосовані до елементів, які відповідають шаблону селектора.
Селектори є одним із найважливіших аспектів CSS, оскільки вони дозволяють націлювати певні елементи на веб-сторінці різними способами, щоб їх можна було стилізувати.

Ми можемо розділити селектори CSS на п’ять категорій:

  • Прості селектори (вибір елементів на основі імені, ідентифікатора, класу)
  • Селектори комбінаторів (вибір елементів на основі певного зв’язку між ними)
  • Селектори псевдокласів (вибір елементів на основі певного стану)
  • Селектори псевдоелементів (вибір і стиль частини елемента)
  • Селектори атрибутів (вибір елементів на основі атрибута або значення атрибута)

Універсальний селектор

Універсальний селектор, позначений зірочкою ( *), відповідає кожному окремому елементу на сторінці.
Універсальний селектор можна опустити, якщо для елемента існують інші умови.
Цей селектор часто використовується для видалення стандартних полів і відступів з елементів для швидкого тестування.

Приклад:

Правила стилю всередині * селектора будуть застосовані до кожного елемента в документі. :

* { margin: 0; padding: 0; }
