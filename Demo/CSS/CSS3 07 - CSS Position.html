CSS Float. Плаваючі елементи

Ви можете переміщати елементи ліворуч або праворуч, але це стосується лише елементів, які генерують прямокутники, які не є абсолютно позиціонованими . Будь-який елемент, який слідує за плаваючим елементом, буде обтікати плаваючий елемент з іншого боку.

Властивість float може мати одне з трьох значень:

  • left - Елемент плаває зліва від блоку, що його містить.
  • right - Елемент плаває з правого боку блоку, що його містить.
  • none - Вилучає властивість float з елемента.
CSS Float

Плаваючі елементи

Плаваючий елемент вилучається з нормального потоку та зміщується ліворуч або праворуч, наскільки це можливо, у просторі, доступному для вмісту елемента. Інші елементи зазвичай обтікають плаваючі предмети, якщо їм не заважає це робити їх clear властивість. Елементи переміщуються горизонтально, що означає, що елемент можна переміщати лише ліворуч або праворуч, а не вгору чи вниз.

Приклад:

У наступному прикладі показано, що зображення має плавати праворуч у тексті:


          img {
            float: right;
          }

Приклад:

У наступному прикладі показано, що зображення мають плавати ліворуч олин за одним:


          ul{
            margin: 0;
            padding: 0;
            list-style: none;
          }
          .thumbnail {
            float: left;
