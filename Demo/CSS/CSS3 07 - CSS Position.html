Позиціонування

Є кілька значень позиціонування, які можна застосувати до елементів. Початкове значення, застосоване до всіх елементів, є «статичним», типом потоку.

Властивість position вказує тип методу позиціонування, який використовується для елемента .
Є п’ять різних значень позиції:

  • static
  • relative
  • fixed
  • absolute
  • sticky

Властивості зміщення позиції:

  • top
  • right
  • bottom
  • left

Існує дві групи властивостей позиціонування:

  • In flow
  • Out-of-flow
CSS Position

Позиція: Static

Елементи HTML за замовчуванням позиціонуються статично. На статичні позиціоновані елементи не впливають властивості top, bottom, left і right.
Елемент з position: static; не позиціонується ніяким особливим чином; він завжди розташовується відповідно до нормального потоку сторінки:

Приклад:


            div.static {
              position: static;
              border: 3px solid #73AD21;
              left: 20px; - не буде застосовуватися
              width: 300px;
              height: 50px;
            }
Цей div елемент має позиціювання: static;

Позиція: Relative

Елемент з position: relative; позиціонується відносно свого нормального положення. Блок змінює своє положення відповідно до властивостей — top або bottom та/або left або right.

Приклад:


          relative {
            position: relative;
            border: 3px solid tomato;
            top: 50px;
            left: 50px;
            width: 150px;
            height: 150px;
          }
Цей div елемент має позиціювання: relative;

Примітка. Елемент можна переміщувати та накладати на інші елементи, але при цьому зберігається простір, спочатку зарезервований для нього в звичайному потоці.
