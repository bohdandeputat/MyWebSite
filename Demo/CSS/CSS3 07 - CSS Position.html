Позиціонування

Є кілька значень позиціонування, які можна застосувати до елементів. Початкове значення, застосоване до всіх елементів, є «статичним», типом потоку.

Властивість position вказує тип методу позиціонування, який використовується для елемента .
Є п’ять різних значень позиції:

  • static
  • relative
  • fixed
  • absolute
  • sticky

Властивості зміщення позиції:

  • top
  • right
  • bottom
  • left

Існує дві групи властивостей позиціонування:

  • In flow
  • Out-of-flow
CSS Position

Позиція: Static

Елементи HTML за замовчуванням позиціонуються статично. На статичні позиціоновані елементи не впливають властивості top, bottom, left і right.
Елемент з position: static;не позиціонується ніяким особливим чином; він завжди розташовується відповідно до нормального потоку сторінки:

Приклад:

Пояснення різних частин:

  • Content (Вміст) – вміст поля, де з’являються текст і зображення;
  • Padding (Внутрішній відступ) – очищає область навколо вмісту. Прозоре поле;
  • Border (Рамка) – межа, що охоплює відступи та вміст;
  • Margin (Зовнішній відступ) – очищає область за межами. Прозоре поле;

Демонстрація моделі за допомогою CSS:

          
          .box {
              padding: 20px;
              background: #7dc765;
          }

Ширина і висота елемента

Щоб правильно встановити ширину і висоту елемента у всіх браузерах, потрібно знати, як працює блокову модель.

Важливо: коли ви встановлюєте властивості ширини та висоти елемента за допомогою CSS, ви просто встановлюєте ширину та висоту області вмісту . Щоб обчислити повний розмір елемента, ви також повинні додати відступи, рамки та поля.

Цей елемент <div> матиме загальну ширину 350 пікселів:

Ось розрахунок:

  • 120 пікселів (ширина)
  • + 20 пікселів (лівий + правий відступ)
  • + 10 пікселів (ліва + права межа)
  • + 0 пікселів (ліве + праве поле)
  • = 150 пікселів

Демонстрація моделі за допомогою CSS:

          
            div {
              width: 120px;
              padding: 10px;
              border: 5px solid gray;
              margin: 0;
            }

Максимальна (мінімальна) Ширина і висота елемента

Щоб правильно встановити максимальну (мінімальну) ширину і висоту елемента до відповідних тегів додають префікси max- (min- ).

Приклад:

  • max-height - задає максимальну висоту елемента
  • max-width - Задає максимальну ширину елемента
  • min-height - Задає мінімальну висоту елемента
  • min-width - Задає мінімальну ширину елемента

Демонстрація моделі за допомогою CSS:

                
                  div {
                    min-width: 150px;
                    max-width: 220px;
                    min-height: 150px;
                    max-height: 220px;  
                    padding: 10px;
                    border: 5px dotted red;
                    margin: auto;
                  }
