Що таке Псевдоклас?

Псевдокласи CSS дозволяють стилізувати динамічні стани елемента, такі як наведення курсора, активний стан і стан фокусу, а також елементи, які існують у дереві документа, але не можуть бути націлені за допомогою інших селекторів без додавання будь-яких Ідентифікатори або класи для них, наприклад, націлені на перший або останній дочірній елемент.

  • Стилізуйте елемент, коли користувач наводить на нього мишу
  • По-різному стилізуйте відвідані та невідвідані посилання
  • Стилізуйте елемент, коли він отримує фокус

Псевдоклас починається з двокрапки (:). Його синтаксис можна подати за допомогою: 


            *selector:pseudo-class { 
              властивість : значення; 
            }

Якірні псевдокласи

За допомогою прив’язки псевдокласів посилання можуть відображатися різними способами:

Приклади:

Ці псевдокласи дозволяють вам відрізняти стиль невідвіданих посилань від відвіданих. Найпоширенішим прийомом стилізації є видалення підкреслень із відвіданих посилань. 


            a:link {
              color: brown;
            }
            
            a:visited {
              color: green;
            }
            
            a:hover {
              color: tomato;
            }
            
            a:active {
              color: aqua;
            }
            
            a:focus {
              color: blueviolet;
            }

Note: Щоб ці псевдокласи працювали ідеально, ви повинні визначити їх у точному порядку —
:link, :visited, :hover, :active, :focus.

Псевдокласи :first-child :last-child :nth-child.

Псевдоклас :first-child відповідає елементу, який є першим дочірнім елементом іншого елемента.

Псевдоклас :first-child відповідає кожному елементу, який є останнім дочірнім елементом іншого елемента.

Приклади:

Селектор ol li:first-child у прикладі нижче вибирає перший елемент списку впорядкованого списку та видаляє верхню межу з нього. 


            li:first-child {
              padding: 10px 10px;
              border-top: 5px dotted red;
              font-weight: bold;
            }

Селектор ol li:last-child у прикладі нижче вибирає останній елемент списку впорядкованого списку та додає до нього ліву межу та фон. 


            div li:last-child {
              padding: 10px 10px;
              background-color: aqua;
              border-left: 5px solid blue;
              font-weight: bold;
            }
