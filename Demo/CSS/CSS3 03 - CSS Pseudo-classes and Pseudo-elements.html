Що таке Псевдоклас?

Псевдокласи CSS дозволяють стилізувати динамічні стани елемента, такі як наведення курсора, активний стан і стан фокусу, а також елементи, які існують у дереві документа, але не можуть бути націлені за допомогою інших селекторів без додавання будь-яких Ідентифікатори або класи для них, наприклад, націлені на перший або останній дочірній елемент.

  • Стилізуйте елемент, коли користувач наводить на нього мишу
  • По-різному стилізуйте відвідані та невідвідані посилання
  • Стилізуйте елемент, коли він отримує фокус

Псевдоклас починається з двокрапки ( :). Його синтаксис можна подати за допомогою:

*selector:pseudo-class { властивість : значення ; }

Якірні псевдокласи

За допомогою прив’язки псевдокласів посилання можуть відображатися різними способами:

Приклад:

Ці псевдокласи дозволяють вам відрізняти стиль невідвіданих посилань від відвіданих. Найпоширенішим прийомом стилізації є видалення підкреслень із відвіданих посилань.

a:link { color: blue; }
a:visited { text-decoration: none; }

Ці псевдокласи дозволяють вам відрізняти стиль невідвіданих посилань від відвіданих. Найпоширенішим прийомом стилізації є видалення підкреслень із відвіданих посилань.

a:hover { color: red; }
a:active { color: gray; }
a:focus { color: yellow; }
