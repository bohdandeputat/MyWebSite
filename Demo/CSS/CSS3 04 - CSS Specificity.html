Що таке специфічність?

Якщо є два або більше правил CSS, які вказують на той самий елемент, селектор із найвищим значенням специфічності «переможе», а його оголошення стилю буде застосовано до цього елемента HTML.

CSS Specificity

Специфічність CSS

Подумайте про специфічність як про оцінку/ранг, що визначає, яке оголошення стилю в кінцевому підсумку буде застосовано до елемента.

Приклад:

У цьому прикладі ми додали селектор класу (під назвою «test») і вказали зелений колір для цього класу. Тепер текст буде зеленим (хоча ми вказали червоний колір для селектора елемента "p"). Це пояснюється тим, що селектору класу надається вищий пріоритет:

          
          .test {color: green;}
          p {color: red;}

CSS Правило !important

Правило !importantв CSS використовується, щоб додати властивості/значенню більшої важливості, ніж зазвичай. Насправді, якщо ви використовуєте !importantправило, воно замінить УСІ попередні правила стилю для цієї конкретної властивості цього елемента!

Приклад:

У прикладі вище. усі три абзаци отримають червоний фоновий колір, навіть якщо селектор ідентифікатора та селектор класу мають вищу специфічність. Правило !importantперекриває background-colorвластивість в обох випадках. 

          
          #myid {
            background-color: blue;
          }

           .myclass {
            background-color: gray;
          }

          p {
            color: red; !important
          }
