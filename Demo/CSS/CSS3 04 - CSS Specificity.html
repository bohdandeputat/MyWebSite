Навігація

Що таке специфічність?

Якщо є два або більше правил CSS, які вказують на той самий елемент, селектор із найвищим значенням специфічності «переможе», а його оголошення стилю буде застосовано до цього елемента HTML.

CSS Specificity

Специфічність CSS

Подумайте про специфічність як про оцінку/ранг, що визначає, яке оголошення стилю в кінцевому підсумку буде застосовано до елемента.

Приклад:

У цьому прикладі ми додали селектор класу (під назвою «test») і вказали зелений колір для цього класу. Тепер текст буде зеленим (хоча ми вказали червоний колір для селектора елемента "p"). Це пояснюється тим, що селектору класу надається вищий пріоритет:

          
          .test {color: green;}
          p {color: red;}
