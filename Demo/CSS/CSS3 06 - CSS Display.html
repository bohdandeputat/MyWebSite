Що таке Потік Документу?

Потік документів – це розташування елементів сторінки, визначене операторами позиціонування CSS, і порядком елементів HTML. Він визначає, як кожен елемент займає простір і як інші елементи розташовуються відповідно.

Normal Flow або Flow Layout — це спосіб відображення блокових і вбудованих елементів на сторінці до внесення будь-яких змін до їх макета.
Потік — це, по суті, набір елементів, які працюють разом і знають один про одного. Щойно щось вилучено з потоку, воно працює незалежно.

Існує три основні способи розташування елементів:

  • Display types
  • Positioning
  • Floats
