Що таке Потік Документу?

Потік документів – це розташування елементів сторінки, визначене операторами позиціонування CSS, і порядком елементів HTML. Він визначає, як кожен елемент займає простір і як інші елементи розташовуються відповідно.

Normal Flow або Flow Layout — це спосіб відображення блокових і вбудованих елементів на сторінці до внесення будь-яких змін до їх макета.
Потік — це, по суті, набір елементів, які працюють разом і знають один про одного. Щойно щось вилучено з потоку, воно працює незалежно.

Існує три основні способи розташування елементів:

  • Display types
  • Positioning
  • Floats

Типи Відображення.

Елементи HTML спочатку позиціонуються за типом відображення. Тип відображення визначає чи можуть інші елементи розташовуватися поруч з ними, і як на це впливають відступи, поля та інші властивості CSS.
Два найважливіші типи відображення:

  • Block
  • Inline

Специфікація CSS визначає значення відображення за замовчуванням для всіх елементів, наприклад, елемент <div> відображається як блок , тоді як елемент <span> відображається в рядку.

Зміна стандартного відображуваного значення

Перевизначення стандартного відображуваного значення елемента є важливим наслідком властивості display.
Наприклад, зміна елемента inline-level для відображення як елемента block-level або зміна елемента block-level для відображення як елемента inline-level.

Перевизначення стандартного відображуваного Inline елемента Display: Block

Вбудований елемент не починається з нового рядка і займає лише необхідну ширину.

Inline elements:

  • <a>
