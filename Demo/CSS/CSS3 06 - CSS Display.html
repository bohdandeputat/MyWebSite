Що таке Потік Документу?

Потік документів – це розташування елементів сторінки, визначене операторами позиціонування CSS, і порядком елементів HTML. Він визначає, як кожен елемент займає простір і як інші елементи розташовуються відповідно.

Normal Flow або Flow Layout — це спосіб відображення блокових і вбудованих елементів на сторінці до внесення будь-яких змін до їх макета.
Потік — це, по суті, набір елементів, які працюють разом і знають один про одного. Щойно щось вилучено з потоку, воно працює незалежно.

Існує три основні способи розташування елементів:

  • Display types
  • Positioning
  • Floats

Типи Відображення.

Елементи HTML спочатку позиціонуються за типом відображення. Тип відображення визначає чи можуть інші елементи розташовуватися поруч з ними, і як на це впливають відступи, поля та інші властивості CSS.
Два найважливіші типи відображення:

  • Block
  • Inline

Специфікація CSS визначає значення відображення за замовчуванням для всіх елементів, наприклад, елемент <div> відображається як блок , тоді як елемент <span> відображається в рядку.

Перевизначення стандартного відображуваного значення елемента є важливим наслідком властивості display.

Дисплей Block

Перевизначення стандартного відображуваного Inline елемента Display: Block

Вбудований елемент не починається з нового рядка і займає лише необхідну ширину.

Inline elements:

  • <span>
  • <a>
  • <img>

Приклади:

          
          span {
              display: block;
          }

          a {
              display: block;
          }

Дисплей Inline

Перевизначення стандартного відображуваного Block елемента Display: Inline

Значення inline властивості display змушує елемент поводитись так, ніби це елемент вбудованого рівня , як <span> або <a> елемент. Правила стилю в наступному прикладі відображають елементи <p> та <li> як елементи вбудованого рівня:

Block elements:

  • <div>
  • <p>
  • <h1> - <h6>

Приклади:

          
          p {
              display: inline;
          }
          ul li {
              display: inline;

Дисплей Inline-Block

Перевизначення стандартного відображуваного Inline-Block елемента Display: Inline-Block

Значення inline-block властивості display змушує елемент генерувати блоковий блок, який буде перетікати з навколишнім вмістом, тобто в тому самому рядку, що й суміжний вміст. Наступні правила стилю відображають елементи як вбудований блок:

Приклади:

          
            div {
                display: inline-block;
            }
            span {
                display: inline-block;
            }

Приховати елемент - display:none або visibility:hidden?

Приховати елемент можна, встановивши displayдля властивості none. Елемент буде приховано, а сторінка відображатиметься так, ніби елемента там немає:

visibility:hidden;також приховує елемент. Однак елемент займатиме стільки ж місця, скільки й раніше. Елемент буде приховано, але все одно вплине на макет:

Приклади:

          
            div {
                visibility: hidden;
            }
            span {
                display: none;
