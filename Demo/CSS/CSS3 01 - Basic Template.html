Основні приклади CSS3

Навігація

Що таке CSS?

  • CSS розшифровується як каскадні таблиці стилів
  • CSS описує, як елементи HTML мають відображатися на екрані, папері чи в інших носіях
  • CSS економить багато роботи. Він може керувати макетом кількох веб-сторінок одночасно
  • Зовнішні таблиці стилів зберігаються у файлах CSS

Синтаксис CSS

Селектор вказує на елемент HTML, який потрібно стилізувати.
Блок декларацій містить одну або кілька декларацій, розділених крапкою з комою.
Кожне оголошення містить назву властивості CSS і значення, розділені двокрапкою.
Кілька декларацій CSS розділені крапкою з комою, а блоки декларацій оточені фігурними дужками.
