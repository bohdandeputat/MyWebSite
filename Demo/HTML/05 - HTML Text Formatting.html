Форматування тексту HTML

HTML містить кілька елементів для визначення тексту зі спеціальним значенням.

Елементи форматування були розроблені для відображення спеціальних типів тексту: 

            b - Жирний текст;
            strong - Важливий текст;
            i - Курсивний текст;
            em - Підкреслений текст;
            mark - Позначений текст;
            small - Менший текст;
            del - Видалений текст;
            ins - Вставлений текст;
            sub - Підрядковий текст;
            sup - Надрядковий текст;

HTML <b> та <strong> Елементи:

Елемент HTML <b> визначає жирний текст без особливого значення.

Елемент HTML <strong> визначає текст із великим значенням. Вміст усередині зазвичай відображається жирним шрифтом.

Приклад:

Цей текст виділений жирним шрифтом

Цей текст важливий!

HTML <i> та <em> Елементи:

Елемент HTML <i> визначає частину тексту альтернативним голосом або настроєм. Вміст усередині відображається курсивом.
Тег часто використовується для позначення технічного терміна, фрази з іншої мови, думки, назви судна тощо./p>

Елемент HTML <em> ввизначає наголошений текст. Вміст усередині відображається курсивом.
Читач з екрану вимовлятиме слова <em> з наголосом, використовуючи словесний наголос.

Приклад:

Цей текст курсивом

Цей текст наголошено

HTML <small>, <mark>, <del>, <ins> Елементи:

Елемент HTML <small> визначає менший текст./p>

Елемент HTML <mark> визначає текст, який слід позначити або виділити.

Елемент HTML <del> визначає текст, який було видалено з документа. Зазвичай браузери наносять рядок через видалений текст.

Елемент HTML <ins> визначає текст, який було вставлено в документ. Зазвичай браузери підкреслюють вставлений текст

Приклад:

Це трохи менший текст.

Не забудьте придбати молоко сьогодні.

Мій улюблений колір - синій червоний.
