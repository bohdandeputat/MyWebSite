Що таке форма HTML?

Форми – важливий елемент багатьох веб-сторінок, оскільки з їх допомогою можна отримати дані, введені користувачами і обробити їх. Форми надають нам значну кількість елементів управління і полів, в які користувач може ввести пошуковий запит, або свій логін і пароль, або заповнити форму для прийому на роботу.

Форма HTML

Форма HTML використовується для збору вводу користувача. Введення даних користувача найчастіше надсилається на сервер для обробки.

Тег <form> використовується для створення HTML-форми.

Для отримання та обробки даних форм використовуються мови веб-програмування, такі як PHP, Node.js.






Форми містять спеціальні елементи, які називаються елементами управління, такими як поле введення, прапорці, перемикачі, кнопки надсилання тощо. Користувачі зазвичай заповнюють форму, змінюючи її елементи керування, наприклад введення тексту, вибір елементів тощо та надсилання цієї форми на веб-сервер для подальшої обробки.

Елемент <label>

Зверніть увагу на використання <label> елемента у прикладі вище.
<label> Тег визначає мітку для багатьох елементів форми.
<label> Елемент корисний для користувачів читання з екрану, тому що екран-читач буде читати вголос етикетку , коли фокус користувача на вхідному елементі.
<label> Елемент також допомагає користувачам , які мають труднощі з клацати на дуже невеликих ділянках (наприклад, радиокнопки і прапорці) - тому що , коли користувач натискає на текст всередині <label> елемента, він перемикає кнопки радіо / прапорець.

Елементи введення

Input Element - Це найбільш часто використовуваний елемент у формах HTML.
Дозволяє вказати різні типи полів введення користувачем залежно від атрибута type. Елемент введення може мати текстове поле типу, поле пароля, прапорець, перемикач, кнопку надсилання, кнопку скидання, поле вибору файлу, а також кілька нових типів введення, представлені в HTML5.

Тег <input> може відображатися різними способами, в залежності від type атрибута:

<input type = "text">
Відображає однорядкове поле введення тексту
<input type = "radio">
Відображає перемикач (для вибору одного з багатьох варіантів)
<input type = "checkbox">
Відображає прапорець (для вибору нуля або більше з багатьох варіантів)
<input type = "submit">
Відображає кнопку подання (для подання форми)
<input type = "button">
Відображає кнопку, на яку можна натиснути

Керування формою

Атрибут formmethod:

Вхідний formmethod атрибут визначає метод HTTP для надсилання даних форми на URL-адресу дії.

Дані форми можуть бути надіслані як змінні URL-адреси (method = "get") або як транзакція повідомлення HTTP (method = "post").

Примітки щодо методу "отримати" (method = "get"):
Цей метод додає дані форми до URL-адреси в парах ім’я / значення
Цей метод корисний для подання форми, коли користувач хоче додати результат до закладки
Існує обмеження кількості даних, які ви можете розмістити в URL-адресі (залежить від браузера), тому ви не можете бути впевнені, що всі дані форми будуть передані правильно
Ніколи не використовуйте метод "get" для передачі конфіденційної інформації! (пароль або інша конфіденційна інформація буде видно в адресному рядку браузера)
Примітки щодо методу "розміщення" (method = "post"):
Цей метод передає дані форми як транзакцію після HTTP
Надіслані форми методом "опублікувати" не можна додавати в закладки
Метод "post" є надійнішим та безпечнішим, ніж "get", а "post" не має обмежень щодо розміру
