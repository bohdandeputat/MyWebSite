Що таке форма HTML?

Форми – важливий елемент багатьох веб-сторінок, оскільки з їх допомогою можна отримати дані, введені користувачами і обробити їх. Форми надають нам значну кількість елементів управління і полів, в які користувач може ввести пошуковий запит, або свій логін і пароль, або заповнити форму для прийому на роботу.

Форма HTML

Форма HTML використовується для збору вводу користувача. Введення даних користувача найчастіше надсилається на сервер для обробки.

Тег <form> використовується для створення HTML-форми.

MND: The Form element

Приклад:

Для отримання та обробки даних форм використовуються мови веб-програмування, такі як PHP, Node.js.






Форми містять спеціальні елементи, які називаються елементами управління, такими як поле введення, прапорці, перемикачі, кнопки надсилання тощо. Користувачі зазвичай заповнюють форму, змінюючи її елементи керування, наприклад введення тексту, вибір елементів тощо та надсилання цієї форми на веб-сервер для подальшої обробки.

Елемент <label>

Зверніть увагу на використання <label> елемента у прикладі вище.
<label> Тег визначає мітку для багатьох елементів форми.
<label> Елемент корисний для користувачів читання з екрану, тому що екран-читач буде читати вголос етикетку , коли фокус користувача на вхідному елементі.
<label> Елемент також допомагає користувачам , які мають труднощі з клацати на дуже невеликих ділянках (наприклад, радиокнопки і прапорці) - тому що , коли користувач натискає на текст всередині <label> елемента, він перемикає кнопки радіо / прапорець.
