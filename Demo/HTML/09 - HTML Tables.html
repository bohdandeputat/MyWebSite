Таблиці HTML

Таблиця HTML дозволяє впорядковувати дані у рядки та стовпці. Вони зазвичай використовуються для відображення табличних даних, таких як списки товарів, дані про клієнта, фінансові звіти тощо.

Створення таблиць в HTML

Елемент HTML <table> представляє дані у вигляді двовимірної таблиці, що складається з рядків і стовпців (клітинок).

  • <tr> - створення рядків таблиці;
  • <td> - створення стовпців всередині рядка таблиці;

Таблиці HTML - Оформлення вигляду таблиць (додавання меж, відступів і згорнутих меж.)

Більшість атрибутів елемента <table>, призначених для оформлення вигляду таблиць у попередніх версіях HTML було видалено в HTML5, тому не використовуйте їх безпосередньо у HTML документі.
Натомість для стилізації таблиць HTML використовуйте CSS атрибути: border, cellpadding, cellspacing, width, align тощо.

Таблиця HTML - Операції з межами, відступи комірок та вирівнювання заголовків

За замовчуванням таблиці не мають меж.

За замовчуванням межі навколо таблиці та їх комірки відокремлені одна від одної.

За замовчуванням текст в комірці вирівнюється по горизонталі по центру комірки.

Щоб змінити налаштування за замовчуванням, ви можете використовувати властивості CSS:

  • border - додавання меж до таблиць;
  • padding - збільшенняя місця навколо вмісту в клітинках таблиці;
  • border-collapse - згорнути меж в таблиці;
  • text-align - вирівнювання вмісту в клітинках таблиці;

Наступні правила стилю додають 1-піксельну межу до таблиці та 10-піксельну відступ до її комірок. 

                <style> 
                    table, th, td {
                        border: 1px solid black;
                      }
                    th, td {
                        padding: 10px;
                      }
                </style>

Наступні правила стилю згортають межі таблиці та вирівнюють текст заголовка таблиці зліва. 

                <style> 
                    table {
                        border-collapse: collapse;
                    }
                    th {
                        text-align: left;
                    }
                </style>

Охоплення стовпців і рядків таблиці HTML

Таблиці HTML можуть мати комірки, які охоплюють кілька рядків і/або стовпців.

Розбиття дозволяє розширити рядки та стовпці таблиці за кількома іншими рядками та стовпцями.
Зазвичай комірка таблиці не може переходити в простір під або над іншою коміркою таблиці.
Але, ви можете використовувати атрибути rowpan або colspan, щоб охоплювати кілька рядків або стовпців у таблиці.

Охоплення декількох стовпців

Щоб охопити в одну клітинку кілька стовпців, використовуйте атрибут colspan

Охоплення декількох рядків

Щоб охопити в одну клітинку кілька рядків, використовуйте атрибут rowspan

Аналогічним чином ви можете використовувати атрибут rowpan для створення комірки, яка охоплює більше одного рядка.
