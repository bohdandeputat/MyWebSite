Списки HTML

Списки HTML використовуються для подання переліку інформації в чітко сформованому та семантичному вигляді.

У HTML є три різні типи списків, і кожен із них має певне призначення та значення.

Невпорядкований список - використовується для створення списку пов’язаних елементів без певного порядку.

Впорядкований список - використовується для створення списку пов’язаних елементів у певному порядку.

Список описів - використовується для створення списку термінів та їх описів.

Невпорядкований список HTML

Невпорядкований список починається з тегу <ul>.
Кожен елемент списку починається з тегу <li>.
Елементи списку за замовчуванням будуть позначені маркерами.

Вкладений невпорядкований список HTML

Створення списоку всередині списку (вкладений списоку).

Ви також можете змінити тип маркера у своєму невпорядкованому списку за допомогою властивості типу CSS list-style.

                    
                      ul {
                        list-style-type: square;    
                        }

Упорядковані списки HTML

Упорядкований список, створений за допомогою <ol> елемента, і кожен елемент списку починається з елемента <li>.
Упорядковані списки використовуються, коли важливий порядок елементів списку.

Впорядкований список

Елементи списку в упорядкованому списку позначені цифрами або буквами.

Списки можуть бути вкладеними (список у списку).

Упорядкований список HTML – атрибут типу

Атрибут type тегу <ol> визначає тип маркера елемента списку:

  • type="1" - Елементи списку будуть пронумеровані цифрами (за замовчуванням);
  • type="A" - Елементи списку будуть пронумеровані великими літерами;
  • type="a" - Елементи списку будуть пронумеровані малими літерами;
  • type="I" - Елементи списку будуть пронумеровані великими римськими цифрами;
  • type="i" - Елементи списку будуть пронумеровані малими римськими цифрами;

Ви також можете змінити тип маркера у своєму невпорядкованому списку за допомогою властивості типу CSS list-style-type.
Наступне правило стилю змінює тип маркера з диска за замовчуванням на roman numbers: 


                    ol {
                        list-style-type: upper-roman;
                       }
