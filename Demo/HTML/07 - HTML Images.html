HTML зображення

Використання графіки робить веб-сторінки візуально привабливими, зображення допомагають краще передати суть і зміст веб-документа. Також, за допомогою HTML-тегів можна робити карти-зображення з активними областями.

Синтаксис зображень HTML

<img> - Тег HTML використовується для вбудування зображення у веб-сторінку.

Тег <img> має два обов'язкових атрибути:
src - Вказує шлях до зображення
alt- Визначає альтернативний текст для зображенн який відображається, якщо браузер не може знайти зображення.

<img src="url" alt="some_text">

Коли веб-сторінка завантажується, браузер отримує зображення з веб-сервера та вставляє його на сторінку.

Якщо зображення розташоване у підкаталозі, необхідно вказати повний шлях до папки та ім’я зображення в src атрибуті:

Елемент <picture> HTML

Елемент HTML <picture> дозволяє відображати різні картинки для різних пристроїв або розмірів екрану..

Синтаксис <picture> HTML

Елемент HTML <picture> надає веб-розробникам більшу гнучкість у визначенні графічних ресурсів.
Елемент <picture> містить один або більше <source> елементів, кожен з яких зі посиланням на різні зображення через srcset атрибут. Таким чином браузер може вибрати зображення, яке найкраще відповідає поточному перегляду та / або пристрою. Кожен <source> елемент має media атрибут, який визначає, коли зображення є найбільш підходящим.

Коли використовувати елемент зображення
Є дві основні цілі picture елемента:

1. Пропускна здатність Якщо у вас невеликий екран або пристрій, не потрібно завантажувати великий файл зображення. Браузер використовуватиме перший source елемент із відповідними значеннями атрибутів та ігноруватиме будь-який з наступних елементів.

2. Підтримка форматування Деякі браузери або пристрої можуть підтримувати не всі формати зображень. Використовуючи picture елемент, ви можете додавати зображення всіх форматів, і браузер використовуватиме перший розпізнаний формат, а також ігноруватиме будь-який із наступних елементів.

Елемент <figure> HTML

Елемент <figure> представляє одиницю вмісту, необов’язково із підписом, який є самодостатнім і на який зазвичай посилаються як на єдиний блок. Зазвичай це зображення, ілюстрація, діаграма або фрагмент коду, на які є посилання в основному тексті.

Синтаксис <figure> HTML

Елемент HTML <figure> визначає самодостатній вміст, як-от ілюстрації, діаграми, фотографії, переліки кодів тощо.
Елемент <figcaption> використовується для додавання підпису для <figure> елемента.
